********** Desktop Group Chat Application  ***************

A desktop-based group chat application developed using **Java Swing**, **Socket Programming**, and **Multithreading**.  
This project allows multiple clients to connect to a server and exchange messages in real time with an Interacted UI like WhatsApp.

## Features
- Multi-client support using multithreading
- Real-time group messaging
- WhatsApp-style chat bubble UI
- Message timestamp
- Client username display
- Easy to run locally

## How to Run the Project

### Step 1: Start Server
- Run `ChatServer.java`

### Step 2: Start Clients
- Run `ChatClient.java`
- Enter username
- Open multiple clients for group chatting
