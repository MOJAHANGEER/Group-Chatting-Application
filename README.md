********** Desktop Group Chat Application  ***************

A Java Swing based desktop group chat application.

## Features
- Multiple clients using multithreading
- WhatsApp-style chat UI
- Socket programming
- MySQL database for message storage

## Technologies Used
- Java (Swing, Socket, Multithreading)
- MySQL
- JDBC

## How to Run
1. Start ChatServer
2. Run ChatClient for each user
3. All users must be on same network
